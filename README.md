## 📝 Sobre o Projeto

O **Site Doces da Vika** é uma landing page e portfólio desenvolvido especialmente para apresentar de forma profissional e atrativa o negócio de confeitaria da minha mãe, Viviane Beiruth. O objetivo principal da plataforma é destacar os seus produtos artesanais (como bolos decorados e doces finos) e facilitar para os clientes o processo de solicitação de orçamentos e contato.

Com foco em uma experiência de usuário (*UX*) sofisticada e um nível de detalhe premium, a interface foi construída para dar protagonismo às fotos dos produtos, passando a sensação de qualidade e carinho presentes nos doces reais.

---

## ✨ Funcionalidades

- **🍰 Visualização de Menu:** Uma lista clara e apetitosa dos doces, bolos e sobremesas disponíveis.
- **🖼️ Galeria de Temas:** Um portfólio visual dedicado a exibir temas de festas passadas, demonstrando a versatilidade e a arte em pasta americana.
- **💬 Solicitação de Orçamento:** Uma secção projetada para direcionar e facilitar o contato direto de clientes interessados em encomendar doces para seus eventos.
- **📱 Responsividade:** O design adapta-se perfeitamente a telemóveis, tablets e desktops.

## 🚀 Como Correr o Projeto Localmente

Se quiseres explorar o código ou executar a aplicação na tua própria máquina, segue os passos abaixo.

### Pré-requisitos
*   [Node.js](https://nodejs.org/en/) (Versão 16+ recomendada)
*   npm (já vem integrado com o Node.js)

### Instalação

1. **Clona este repositório:**
   ```bash
   git clone https://github.com/BeiruthDEV/site-doces-da-vika.git
   ```

2. **Acede ao diretório do projeto:**
   ```bash
   cd site-doces-da-vika
   ```

3. **Instala as dependências:**
   ```bash
   npm install
   ```

4. **Inicia o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

5. **Abre no navegador:**
   O terminal irá fornecer um link local (frequentemente `http://localhost:5173/`). Clica nele ou copia para o teu browser favorito.
